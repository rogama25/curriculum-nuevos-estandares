UBUVoiceAssistant

UBUVoiceAssistant es un asistente de voz programado en Python basado en un asistente open-source llamado Mycroft
Para ofrecer las funcionalidades deseadas se crearon 5 skills disponibles en esta cuenta de Github.
El asistente se puede instalar en cualquier dispositivo con sistema operativo Linux y se puede interactuar con él mediante voz y texto.
El instalador y la documentación se pueden encontrar en este repositorio.
También se ofrece soporte para instalarlo en Windows usando WSL2.
En un futuro se podrá instalar en dispositivos Android, iOS, MacOS y los dispositivos propios de Mycroft, como el Mark II.

Monitorización de sensores

El programa de monitorización de sensores permite monitorizar valores de varios ordenadores (que se comportan como sensores) de manera centralizada desde otro equipo.
Se ha utilizado Docker para facilitar el despliegue de los sensores y el cliente de monitorización.
Los sensores están programados con Flask, incluyendo un sistema de autenticación.
